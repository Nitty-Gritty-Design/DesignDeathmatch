# VEKTRA Logo Concept

## Mark Geometry
The logo mark combines a simplified sine wave (3 peaks, representing signal/frequency) with a diagonal vector arrow (pointing 2 o'clock, representing velocity/vector), merged into a single geometric form.

## Logotype Treatment
Logotype uses Space Mono Bold, uppercase, with tight kerning (tracking: -20). Letters are aligned to the baseline of the mark, creating a cohesive unit.

## Mark-Logotype Relationship
The mark sits 16px to the left of the logotype, aligned at the baseline. The mark is 32px tall, logotype is 28px tall (aligned to mark's vertical center).

## Animation Plan
The logo animation starts with a single dot (cursor blink) at the start of the wave, draws the sine wave in 1.2s using stroke-dasharray, then draws the arrow in 0.8s, then the logotype fades in over 0.5s. Total duration: 2.5s, no loop.

## Size Verification
- 240px wide: Full detail, mark and logotype clear
- 80px wide: Mark simplified to 3 wave peaks, logotype 12px size, still legible
- 32px wide: Mark reduced to 2 wave peaks + arrow, logotype removed, mark only (logo-mark.svg) — holds at 32px, no loss of clarity
