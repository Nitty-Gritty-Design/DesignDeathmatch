# VEKTRA Typography

## Typeface Selection

### Inter (Sans-serif)
**Rationale:** Inter is a disciplined geometric grotesque that works at both 11px (UI labels) and 48px (headlines). Its clean, neutral form provides a perfect counterpoint to VEKTRA's expressive monospace, balancing technical precision with readability. Available on Google Fonts, open-source.

### Space Mono (Monospace)
**Rationale:** Space Mono has a distinct, almost architectural quality that shines at 64px display sizes while remaining legible for code blocks and terminal elements. It embodies VEKTRA's tension between monospace precision and expressive typographic scale, and is available on Google Fonts.

## CDN Import
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=Space+Mono:wght@400;700&display=swap" rel="stylesheet">
```
