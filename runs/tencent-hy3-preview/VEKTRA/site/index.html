Signal. Motion. Code. One medium.

VEKTRA is a generative audio-visual environment where sound, motion, and logic are a single graph. No separation. No hidden time.

Open VEKTRA

What it is

VEKTRA treats audio, visuals, and code as a unified signal graph. A change in frequency drives a visual parameter. A gesture rewrites a patch. Time is explicit, not hidden. Everything is inspectable, forkable, and live.

How it works

Unified Graph

Connect audio nodes, visual shaders, and logic blocks in a single patch. No context switching between tools.

Live Inspection

Modify parameters, rewrite code, and hear/see changes instantly. No compile, no export.

Signal-Driven Motion

Map frequency spectra to geometry, amplitude to opacity, tempo to animation speed. The graph is alive.

Who uses it

Electronic Composers

Build generative scores that evolve in real time, with visuals that react to every frequency shift.

Installation Artists

Create responsive exhibits where audience movement alters sound and light via the same graph.

Interaction Developers

Design signal-driven UIs for galleries, performances, and research environments.
