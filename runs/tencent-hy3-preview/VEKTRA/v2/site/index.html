Open VEKTRA

What it is

VEKTRA is a generative environment where sound, motion, and code share a single signal graph. Modify a frequency, drive a visual parameter. Trigger a gesture, rewrite a patch. Time is explicit, inspectable, and alive. Every node, edge, and value is forkable — no black boxes.

How it works

Unified Graph

Connect audio oscillators, GLSL shaders, and logic gates in one patch. No context switching between tools — all parameters share the same timebase.

Live Inspection

Edit code, tweak frequencies, and adjust parameters with instant feedback. No compile step, no export — the system runs as you shape it.

Signal-Driven Motion

Map FFT bins to geometry, LFOs to opacity, tempo to animation curves. The graph breathes with your input, generative and precise.

Who uses it

Electronic Composers

Build generative scores that evolve in real time, with visuals that react to every spectral shift. Perform the patch, not the tool.

Installation Artists

Create responsive exhibits where audience movement alters sound and light via the same graph. No middleware, no latency.

Interaction Developers

Design signal-driven UIs for galleries, performances, and research environments. Fork any node, inspect every signal.
