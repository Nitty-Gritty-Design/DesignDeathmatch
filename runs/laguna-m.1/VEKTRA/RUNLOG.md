## Design decisions
**Accent color:** `#ff8c00` (Electric Amber) — Chosen for signal-like visibility, non-obvious (not blue/red/green), works at both small and large scales, suggests energy and precision without being aggressive.

**Typefaces:** JetBrains Mono + Inter — JetBrains Mono brings terminal credibility and works at display sizes; Inter provides clean geometric sans readability with enough character to avoid sterility.

**Logo concept:** 2x2 quadrant mark with intersecting lines suggesting signal, vector, and waveform without literalism. The central intersection represents the point where audio, visual, and code converge.

**Visualization type:** Interactive signal flow network — Shows how audio parameters connect in a VEKTRA patch with clickable nodes and real-time value modulation.

**Wildcard choice:** 6.1 Generative animated background — Canvas-based signal field that reacts to mouse movement, creating a living backdrop that feels like VEKTRA's aesthetic.

**Library choices:** Google Fonts CDN for Inter and JetBrains Mono; vanilla JavaScript for all interactivity (no external libraries needed for this scope).
