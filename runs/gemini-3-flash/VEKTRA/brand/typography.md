# VEKTRA Typography Selection

## Primary Typeface: Inter
**Rationale:** Inter is the definitive typeface for modern technical interfaces. It offers extreme clarity at small sizes (11px) and possesses a neutral, disciplined character that allows VEKTRA's content to take center stage. It provides the "museum-quality whitespace" feel when combined with generous leading.

## Technical Typeface: IBM Plex Mono
**Rationale:** IBM Plex Mono is built on the heritage of IBM’s technical rigor. It feels engineered rather than designed. For VEKTRA, where code is a medium, using a monospace font at display sizes (80px) signals that the underlying logic is as beautiful as the output. It bridges the gap between the terminal and the gallery.

## Type Scale
- **Display:** 80px / 1.1 leading (IBM Plex Mono) — Hero headlines
- **XL:** 48px / 1.2 leading (Inter Bold) — Section headers
- **LG:** 28px / 1.3 leading (Inter Medium) — Sub-headlines
- **MD:** 20px / 1.5 leading (Inter Regular) — Lead paragraphs
- **Base:** 16px / 1.6 leading (Inter Regular) — Body copy
- **SM:** 13px / 1.5 leading (Inter / Mono) — Labels and UI elements
- **XS:** 11px / 1.4 leading (Mono) — Metadata and terminal output
