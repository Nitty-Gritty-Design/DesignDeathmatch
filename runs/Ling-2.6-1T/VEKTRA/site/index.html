A software environment where sound, motion, and code converge as one signal.

VEKTRA treats time as an explicit parameter — every oscillator, envelope, and transform is visible, editable, and forkable at any moment. Build things that behave, not just things that look.

What It Is

VEKTRA is not a DAW. It is not a creative suite. It is not a plugin. It is closer to a language — a way of thinking about time, signal, and form simultaneously.

Founded by a sound engineer and a computer vision researcher who met at an artist residency in Leipzig in 2019, VEKTRA emerged from the need for tools that don't separate the visual from the sonic from the logical. In VEKTRA, a change in frequency can drive a visual parameter. A gesture can rewrite a patch. Time is explicit, not hidden. Everything is inspectable, forkable, and live.

How It Works

Signal Graph Architecture

Nodes connect through a unified graph where audio, visual, and control signals flow bidirectionally. No separate tracks, no hidden layers.

Live Code Environment

Edit running patches in real-time. Every change takes effect immediately without stopping the signal. Fork any state to explore alternatives.

Parametric Time

Time is a first-class parameter. Scrub, warp, and reverse any signal independently. Create non-linear compositions with explicit temporal relationships.
