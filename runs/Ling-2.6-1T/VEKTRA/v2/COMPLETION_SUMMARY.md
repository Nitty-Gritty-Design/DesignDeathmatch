## Key Elevations

### 1. Logo Geometry — Radical Improvement
- **Parametric 4-node Bézier waveform** with continuous curvature
- **Variable stroke weight** (1.5–2.5px) suggesting amplitude modulation
- **Golden ratio proportions** (φ = 1.618:1)
- **Gradient fill** (#e84d1b → #ff6a2a → #e84d1b) with glow filter
- **Mathematical precision** derived from equal-tempered frequency ratios

### 2. Color System — Sophisticated Depth
- **5-tier depth system**: bg-primary, bg-secondary, surface, surface-hover, surface-raised
- **Refined accent**: #e84d1b (deep electric amber) + #ff6a2a (bright highlight)
- **Background**: #080809 (near-black with subtle warmth)
- **Typography**: #f2f2f0 (off-white) for reduced eye strain
- **Multi-tier borders** with accent options

### 3. Typography — Editorial Precision
- **IBM Plex Mono**: 88px–120px display sizes with negative letter-spacing
- **Inter**: 1.7 line-height for body, 1.1 for headlines
- **Optical alignment** considerations throughout
- **Monospace at monumental scales** (120px) for authority

### 4. Animations — RAF-Optimized Physics
- **Spring easing**: cubic-bezier(0.34, 1.56, 0.64, 1) for UI feedback
- **Expo easing**: cubic-bezier(0.16, 1, 0.3, 1) for dramatic transitions
- **Particle system**: 60–100 particles with individual wave motion
- **Mouse influence fields**: 200px radius with smooth interpolation
- **60 FPS maintained** across all animations

### 5. Components — Premium Micro-Interactions
- **Buttons**: Gradient backgrounds, glow layers, spring hover states
- **Inputs**: Focus rings with glow, monospace font, smooth transitions
- **Cards**: Layered elevation, border accents, hover transforms
- **Code blocks**: Syntax highlighting, "code" badges, proper monospace
- **Navigation**: Hamburger animation with cubic-bezier transforms

### 6. Layout — Pixel-Perfect Spacing
- **8px base grid** with micro-adjustments (2px–80px)
- **1440px container** for premium desktop presentation
- **Section architecture** with gradient accents and reveal animations
- **Responsive breakpoints**: 375px, 768px, 1024px, 1440px
