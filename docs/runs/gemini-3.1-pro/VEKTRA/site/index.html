01 / System

The visual is not an afterthought. The sonic is not an overlay.

VEKTRA is a unified graph environment where visual parameters are directly modulated by audio frequencies. It is not a timeline. It is not a canvas. It is a live organism constructed from logic nodes, processing time and signal continuously. Everything is inspectable, forkable, and alive.

02 / Architecture

Unified Graph Topology

Audio oscillators, GLSL shaders, and structural logic coexist on the same node plane. Connect a low-frequency oscillator to camera field-of-view seamlessly.

Sample-Accurate Time

Time is explicit. Timing logic is computed at the audio sample rate, not the frame rate, preventing phase issues between sound and visual reaction.

Forkable States

Save the entire state of the machine at any millisecond. Branch compositions iteratively and script new node types on the fly in the built-in terminal.
