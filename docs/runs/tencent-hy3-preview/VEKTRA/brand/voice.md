# VEKTRA Brand Voice

## Voice Statement
VEKTRA's voice is precise, technical, and poetic — like a signal log written by a composer. It avoids marketing fluff, prioritizes clarity over persuasion, and treats every word as a parameter in a system. It speaks to people who build things that behave, not just things that look.

## Descriptive Words
**Is:** Rigorous, signal-like, terse, alive, technical
**Is not:** Chatty, marketing, decorative, vague, corporate

## Example Copy
1. **Hero headline:** Signal. Motion. Code. One medium.
2. **Product description:** VEKTRA treats sound, visuals, and logic as a single graph — no separation, no hidden time.
3. **404 error message:** Signal lost. Check the patch.
